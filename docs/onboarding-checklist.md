# FINPAL™ Accounting — Client Onboarding Checklist

## Step 1: Sign-Up (Day 1)
- [ ] Client completes sign-up form at finpal.online/accounting
- [ ] Payment confirmed (R350 debit order or EFT)
- [ ] Welcome email sent automatically

## Step 2: Setup (Day 1–2)
- [ ] Create client profile in FINPAL™ system
- [ ] Set up cloud accounting workspace
- [ ] Send client their portal login
- [ ] WhatsApp intro message sent

## Step 3: Data Collection (Day 2–5)
Client sends us:
- [ ] Last 3 months bank statements (PDF)
- [ ] Any outstanding invoices
- [ ] List of recurring expenses
- [ ] VAT registration number (if applicable)
- [ ] Business registration documents

## Step 4: First Month Processing
- [ ] Import bank statements
- [ ] Categorise all transactions
- [ ] Reconcile accounts
- [ ] Prepare income statement
- [ ] Prepare balance sheet
- [ ] VAT calculation (if applicable)
- [ ] Human review & sign-off
- [ ] Reports delivered to client portal

## Step 5: Ongoing (Monthly)
- [ ] Automated reminder sent to client on 1st of month
- [ ] Client submits docs by 3rd
- [ ] Processing complete by 6th
- [ ] Reports delivered by 7th
- [ ] Client sign-off requested
